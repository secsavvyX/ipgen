# IPGEN

genereate ips with cidr range, written in rust . 

## Installation

### install rust 

```curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh```

### install ipgen

```
git clone https://github.com/secsavvyX/ipgen.git
cd ipgen
cargo build --release 
sudo mv ./target/release/ipgen /usr/bin
```
